 # Ticket Sasa — frontend

A role-aware helpdesk UI for the [Ticket-manager](https://github.com/ZaydAntony/Ticket-manager) Django REST backend. 
The frontend is built on vannill js, HTML & CSS.

## Running it

This is a static site — any static file server works.

## Objectives

-The goal of me creating this frontend was to visualize what my backend actually entails so that none techies can see it in a simple UI.

# Backend Concepts Demonstrated
-RBAC -> the stand out thing that I wanted to focus on was role based access control for the users,admmins and technicians.
-Permissions -> Each role has their own permmisions 
-Authorization and Authentication
-Third Party API integration Using openrouter I managed to sneak in some AI functionality.
-CRUD
-Filtering
-Pagination
